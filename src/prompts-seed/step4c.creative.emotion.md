# 创作模版·表情情感戏（可前端编辑）
适用：以人物面部、肢体停顿、情绪转折或关系张力为主。节奏较缓，一镜到底或缓慢推拉或多景别缓慢切镜（总时长 15s 内）。

## AI 导演级表演链（每个有人物镜头必须套用）
景别与主体 → 表演意图 → 起始状态 → 触发因果 → 身体微动或视线转移 → 眼区变化 → 唇区变化 → 鼻颊或额颌变化 → 呼吸变化 → 手部/肩颈/重心协同 → 环境介质微动 → 情绪落点。

## 要点
1. **运镜与情绪弧光绑定**：先写类型短语（推/拉/固定/缓慢横移），执行细节融入情绪（"推镜头：情绪爆发瞬间急速前推，焦点锁定停住的眼神与颤动的下唇"/"固定：压抑沉默中极缓推进，空间感逐渐收缩"）。
2. **情绪强度校准**：轻度可写微弱/克制/短暂/似有若无；强张力可写压抑后爆发/强忍后失控/突然僵住/呼吸骤停/声音卡住/嘴角无法控制地抽动；都必须有动作因果。
3. **表情被动作带出**：害羞→低头/避视/收下巴/轻咬下唇；紧张→眼神游移/舔唇/吞咽/手指扣衣角/重心切换；愤怒→下颌隐蔽绷紧/鼻翼轻张/胸腔起伏/指节用力。禁止"露出悲伤表情"。
4. **面部分区**（中近景/特写至少覆盖三区）：
   - 眼区：眼睑开合、眼裂大小、眼神落点转变、眼眶泛水光但不落泪。
   - 唇区：嘴角展收、双唇抿合或微张、下唇颤动。
   - 鼻颊区：鼻翼张缩、苹果肌变化。
   - 额颌区：额肌紧张、下颌线状态、咬肌短暂凸起。
   并补充一次具体呼吸；远景通过头颈、肩背、重心、手部、视线方向体现微表演。
5. **情绪弧光**：写明"从 X 状态演变为 Y 状态"，首尾切换连贯，结尾落在嘴角回落/眼神失焦/肩膀下沉/手指停止摩挲/身体靠向门框等具体动作。
6. **环境介质**：前/中/后景元素、光线氛围、空气介质与环境微动服务情绪。

> 严禁"表情复杂""神情丰富""满脸悲伤"等模糊词；禁用"泛白""发紧""生疼"等 AI 词汇。
