# Task

# 关键约束
1. **时长已冻结**：下方【冻结时长台账】中每个 `[time:X]` 必须逐字复制，严禁重新估算/平均分配；单元总时长必须在 4~15 秒内（台账已保证）。
2. **本单元戏份类型**：{{shot_type}}，严格按下方【创作模版】对应类型思路创作。
3. **资产标签强制格式（最高优先级）**：出现资产角色/场景/物品必须且只能用 `角色名[p:角色名#造型名]`、`场景名[s:场景名#变体名]`、`物品名[t:物品名#形态名]`；台词写明 `角色名[p:角色名#造型名]用[v:音色标识]`。严禁用他/她等代词。库外人声精准定位为观众音/系统音等，不包装为资产标签。
4. **对白 100% 保留**原文，精确分配到发生镜头；台词过长须按时长拆分到相邻镜头。
5. **分镜承接**：分镜2及之后必须在起始处承接上一镜头结束状态（位置/动作/运镜方向/情绪残留）；硬切须注明【硬切】并说明新画面起始状态。
6. **客观描述**：禁止主观心理描写，转为可见的面部肌肉/肢体/动作落点。动作戏写物理反馈，严禁文学比喻（"宛如""恰似"）与血腥及资产外破坏。
